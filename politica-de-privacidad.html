Política de Privacidad

Última actualización: Mayo 2026

01. Información que recopilamos

La aplicación "Noticias con Rocimar Morales" no recopila información personal identificable de sus usuarios. La app almacena localmente en el dispositivo el historial de relatos generados y preferencias de uso.

02. Uso de la información

Los datos almacenados localmente se usan únicamente para mejorar la experiencia del usuario dentro de la aplicación.

03. Servicios de terceros

La app utiliza la API de Anthropic (Claude) para generar resúmenes y relatos periodísticos. Los textos de las noticias son enviados a esta API para su procesamiento. No compartimos datos personales con terceros.

04. Contacto

Para cualquier consulta sobre esta política, puedes contactarnos en: salasclarae@gmail.com
